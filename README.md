# Задание 1

## Задача 4 

### Текст задачи

Дерево содержит строки. Заменить в каждой первый символ на заданный.

### Алгоритм решения

1.	Запросить у пользователя количество строк для ввода, считав данные из консоли и преобразовав их в целое число.
2.	Сформировать бинарное дерево поиска. Для этого создать последовательность seq { 1 .. count } и применить к ней агрегирующую функцию Seq.fold с начальным значением Empty (пустое дерево). На каждом шаге цикла запрашивать у пользователя новую строку и добавлять её в дерево с помощью рекурсивной функции insert, которая распределяет элементы по ветвям согласно алфавитному порядку.
3.	Запросить у пользователя символ для замены. Считать строку и извлечь из неё только первый символ, обратившись к нему по индексу .[0].
4.	Вывести исходное построенное дерево в консоль с помощью рекурсивной функции printTree, которая наглядно визуализирует структуру ветвей (корень, левое и правое поддеревья) за счет отступов пробелами.
5.	Применить к исходному дереву рекурсивную функцию обхода treeMap, передавая ей лямбда-выражение fun s -> if s.Length > 0 then string ch + s.[1..] else s. Это преобразование обойдет каждый узел и, если строка внутри не пустая, заменит ее первый символ на заданный (склеив новый символ ch и срез оригинальной строки s.[1..]).
6.	Вывести сформированное новое дерево в консоль с помощью функции printTree, чтобы продемонстрировать результат замены.
