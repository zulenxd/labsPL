# Назаренко Артём ИТС-1 Лабораторная №8

# Задание 1

## Задача 4 (Меню кафе)

### Текст задачи

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область,
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных
типов данных). Приложение должно выполнять следующие функции:

1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно).
  
2 запроса должны возвращать перечень, 2 запроса одно значение.
В классе должны присутствовать свойства, конструкторы, перегруженный метод ToString(). Весь
функционал приложения реализовать в виде методов вспомогательного класса с помощью LINQ-
запросов.
Предусмотреть обработку возможных ошибок при работе программы.
Пункт 5 оценивается в 2 балла, остальные пункты по 1 баллу. Максимально за лабораторную работу
можно получить 6 баллов.

### Алгоритм решения

1. Попадаем в точку входа консольного приложения. Инициализируется бесконечный цикл `while` с выводом главного меню кафе.
2. Пользователь вводит название, категорию, цену и статус вегетарианского блюда. Считываем их из консоли с немедленной проверкой корректности в блоках `try-catch` через статические методы `MenuItem.Validate...`.
3. После успешного ввода без ошибок вызывается конструктор `MenuItem(...)` для создания нового объекта блюда.
4. Для созданного объекта вызывается метод `MenuManager.AddMenuItem()`, который автоматически назначает ему уникальный Id (на основе максимального существующего) и добавляет в список.
5. Вызывается метод `MenuManager.SaveToFile()`, который с помощью класса `BinaryWriter` сериализует весь список блюд и сохраняет его в бинарный файл `menu.bin`.
6. Пользователь выбирает пункты меню для выполнения запросов (4–7) или просмотра списка (1). Вызывается метод `MenuManager.LoadFromFile()`, который через `BinaryReader` десериализует данные из файла обратно в список объектов.
7. При выполнении запросов запускаются LINQ-выражения и методы (`Where`, `OrderBy`, `Average`, `Count`), которые фильтруют блюда по категории/статусу, сортируют их по цене или высчитывают агрегатные значения.
8. Вывод результатов работы программы, списков блюд (через переопределенный метод `ToString()`) и предупреждений о неверном вводе в окно консоли.
