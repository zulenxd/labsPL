# Задание 1

## Задача 4

### Текст задачи

Составить программу вычисления наибольшего общего делителя двух
натуральных чисел

### Алгоритм решения

1. попадаем в go
2. запрашиваем у пользователя первое натуральное число
3. считываем введенное значение в переменную Num1
4. запрашиваем и считываем второе натуральное число в переменную Num2
5. проверяем что введенные числа больше нуля (Num1 > 0, Num2 > 0)
6. запускается nod(Num1, Num2, Result)
7. внутри nod рекурсивно вычисляется остаток от деления первого числа на второе с помощью mod
8. рекурсивные вызовы продолжаются, пока второе число не станет равным нулю
9. при срабатывании базового случая вычисленный НОД записывается в переменную Result

# Задание 2

## Задача 4

### Текст задачи

В списке натуральных чисел подсчитать их количество, оканчивающихся
заданной цифрой

### Алгоритм решения

1. попадаем в go
2. запрашиваем у пользователя список натуральных чисел
3. считываем ввод пользователя в переменную List
4. запрашиваем и считываем искомую последнюю цифру в переменную Digit
5. проверяем корректность введенного списка с помощью is_list
6. запускается count_digits(List, Digit, Result)
7. внутри count_digits список рекурсивно разделяется на голову (head начальный элемент) и хвост (tail оставшуюся часть списка)
8. вычисляется остаток от деления головы на 10 (H mod 10) и сравнивается с искомой цифрой D
9. если цифры совпадают, программа уходит в рекурсию для хвоста, а при возврате увеличивает счетчик на 1
10. если цифры не совпадают, программа продолжает рекурсивный обход хвоста без добавления 1 к счетчику
11. при достижении пустого списка рекурсия останавливается, формируя итоговую сумму в переменной Result
12. выводим результатw
