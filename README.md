# Задание 1

## Задача 4

### Текст задачи

Получить список из длин строк, содержащихся в исходном списке

### Алгоритм решения

1. попадаем в main
2. спрашиваем у пользователя, сколько он хочет ввести слов
3. считываем введенное количество (count)
4. запускается функция getWords
5. слова будут добавляться в список до тех пор, пока count не будет = 0
6. с помощью List.map String.length выводим длину элементов списка

# Задание 2

## Задача 4

### Текст задачи

Найти сумму тех элементов списка, в которых встречается заданная цифра.

### Алгоритм решения

1. попадаем в main
2. спрашиваем у пользователя, сколько он хочет ввести чисел
3. считываем введенное количество (count)
4. запускается функция getNumbers
5. слова будут добавляться в список до тех пор, пока count не будет = 0
6. запрашиваем цифру, которую нужно будет найти в элементах списка
7. с помощью List.fold выводим сумму элементов списка, содержащих нужную цифру
