## Задача 4

### Текст задачи

Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий минимальное из полей. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения

1. попадаем в main
2. запрашиваем у пользователя три целых числа и считываем их в переменные a, b, c
3. вызывается конструктор ThreeNumbers(a, b, c) для создания базового объекта nums1
4. запускается метод nums1.FindMinimum(), который сравнивает поля _first, _second, _third и возвращает наименьшее
5. с помощью конструктора копирования ThreeNumbers(nums1) создается копия базового объекта в переменной nums2
6. запрашиваем у пользователя габариты коробки и материал
7. вызывается конструктор дочернего класса Box(width, height, depth, material)
8. для объекта box1 запускаются методы CalculateVolume(), CalculateSurfaceArea() и IsCube(), вычисляющие физ характеристики
9. с помощью конструктора копирования Box(box1) создается копия коробки в переменной box2
10. вывод результатов

# Задание 2 и 3

## Задача 4

### Текст задачи

Название класса: RightTriangle; поля: double a, double b; Вычислить площадь треугольника.
Результат должен быть типа double

Унарные операции:
++ увеличивает стороны треугольника в 2 раза (чтобы вновь получить
катеты прямоугольного треугольника);
-- уменьшает стороны треугольника в 2 раза (чтобы вновь получить
катеты прямоугольного треугольника);
Операции приведения типа:
double (явная) – результатом является площадь треугольника, если
треугольник существует и отрицательное число в противном случае;
bool (неявная) – результатом является true, если треугольник с такими
длинами сторон существует и falseв противном случае.
Бинарные операции:
<= Trianglet1, Trianglet2 – сравнивает площади треугольников;
>= Trianglet1, Trianglet2 - сравнивает площади треугольников.


### Алгоритм решения

1. попадаем в main
2. создается треугольник tDefault с помощью конструктора по умолчанию (устанавливаются катеты 1.0 и 1.0)
3. запрашиваем у пользователя длины катетов для первого треугольника (a1 и b1); считывание идет через ReadPositiveDouble, который в цикле проверяет, что введено именно положительное число
4. пытаемся создать объект t1 внутри блока try-catch, значения передаются в конструктор, где свойства A и B проверяют, что числа больше нуля
5. выводим на экран данные t1 и вычисляем его площадь с помощью CalculateArea
6. вызываем перегруженный унарный оператор ++ (t1++), внутри которого создается и возвращается новый треугольник с катетами, увеличенными в 2 раза
7. вызываем перегруженный унарный оператор -- (t1--), внутри которого создается и возвращается новый треугольник с катетами, уменьшенными в 2 раза
8. выполняется явное приведение t1 к типу double: программа проверяет, существует ли треугольник, и возвращает вычисленную площадь
9. выполняется неявное приведение t1 к типу bool: программа проверяет катеты и возвращает true, так как оба катета больше нуля
10. запрашиваем у пользователя длины катетов для второго треугольника (a2 и b2) и аналогично создаем объект t2 с проверкой на ошибки
11. с помощью <= и >= сравниваем площади t1 и t2, после чего выводим результат (true или false) на экран
