## Задача 4

### Текст задачи

Сформировать список из последовательно вводимых слов.

### Алгоритм решения

1. попадаем в main
2. запускается рек.функция spisok
3. считываем введенное слово
4. если пользователь вводит 0, то программа выведет готовый список
5. если не 0, то добавляем слово к списку
6. выводим список

# Задание 2

## Задача 4

### Текст задачи

Найти максимальную цифру натурального числа.

### Алгоритм решения

1. попадаем в main
2. запускается функция res (result)
3. считываем введенную строку
4. запускается функция maxcifra и перевод строки в целое число под модулем 
5. если пользователь вводит число < 10, то программа выведет введенное число
6. если число > 10, то рекурсивно сравнивает посл цифру n/10 c n%10 и берет наибольшее с помощью max 
7. выводим максимальную цифру
