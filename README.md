# Назаренко Артём ИТС-1 Лабораторная №9

# Задание 1

## Задача 4

### Текст задачи

Реализовать приложение из ЛАБОРАТОРНОЙ РАБОТЫ №6 ЗАДАНИЯ 2 и 3 (Перегрузка операций) в
виде WPF-приложения с графическим интерфейсом. Все требования к функционалу сохраняются.
Предусмотреть обработку возможных ошибок при работе программы.
Максимально за лабораторную работу можно получить 6 баллов.
Коды, оформленные с нарушением код-стайла, или реализованные с помощью LINQ не принимаются!

### Алгоритм решения

1. попадаем в точку входа WPF приложения. инициализируется графическое окно MainWindow
2. пользователь вводит длины катетов для первого прямоугольного треугольника в поля ввода "Катет а" и "Катет b". считываем их из текстовых полей в переменные a и b
3. при нажатии кнопки "Создать t1" вызывается конструктор RightTriangle(a, b) для создания базового объекта _t1
4. для объекта _t1 через элементы управления запускаются перегруженные унарные операторы ++ и --, которые возвращают новые объекты с увеличенными или уменьшенными в 2 раза катетами
5. вызываются перегруженные операции приведения типа для объекта _t1. явная (double) для вычисления и получения площади и неявная (bool) для проверки существования треугольника
6. пользователь вводит длины катетов для второго прямоугольного треугольника в поля ввода "Катет а" и "Катет b". вызываем конструктор RightTriangle(a, b) для создания объекта _t2
7. при нажатии кнопки "Сравнить" запускаются перегруженные бинарные операторы <= и >=, которые вызывают метод CalculateArea() и сравнивают площади объектов _t1 и _t2
8. вывод результатов работы программы в окно логов
